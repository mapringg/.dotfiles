# Dotfiles Repo

## Commit Types

When committing in this repo, use these types instead of the standard conventional commit types:

| Type   | Description                                    |
| ------ | ---------------------------------------------- |
| add    | Add a new tool, plugin, or config              |
| remove | Remove a tool, plugin, or config               |
| update | Update existing configuration or settings      |
| fix    | Fix a broken config or incorrect setting       |
| style  | Formatting or cosmetic changes to config files |
| chore  | Maintenance (gitignore, stow, Makefile, etc.)  |
| docs   | Documentation changes                          |
