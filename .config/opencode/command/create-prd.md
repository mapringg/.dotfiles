Convert my feature requirements into structured PRD items. Write to PRD.json.

Use this exact format:
```json
[
  {
    "category": "architecture|integration|unknown|feature|polish",
    "description": "What this item accomplishes",
    "steps": ["Step to verify 1", "Step to verify 2"],
    "passes": false
  }
]
```

Be specific about acceptance criteria. Each item should be small enough for one iteration.
